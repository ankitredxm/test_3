# 🎓 Student Management System (Java Console Application)

A simple Java console-based application to manage student records.

This project allows users to:
- ➕ Add a student
- 📋 View all students
- 🔍 Search for a student by ID
- 🗑️ Delete a student

---

## 🚀 Features

- Object-Oriented Design
- Clean console UI
- CRUD Operations
- Input validation
- Easy to extend

---

## 🛠 Technologies Used

- Java (JDK 8+)
- VS Code
- Git & GitHub

---

## 📂 Project Structure

```
StudentManagementSystem/
│
├── Student.java
├── StudentManager.java
└── Main.java
```

---

## ▶️ How to Run

### 1️⃣ Compile the project

Open terminal inside the project folder and run:

```
javac Main.java
```

### 2️⃣ Run the application

```
java Main
```

---

## 📖 How It Works

- `Student` class → Represents a student entity
- `StudentManager` class → Handles business logic
- `Main` class → Handles user interaction

Data is stored temporarily using an ArrayList.
